## Build tech stack
- Hugo
- GitHub Actions
- Tailwind CSS
- JavaScript
- HTML

## Installation
1. Clone the repository
2. Check out [Hugo's installation guide](https://gohugo.io/getting-started/installing/) for your OS
2. Run `hugo server`
3. Visit `localhost:1313` in your browser
4. Enjoy!
