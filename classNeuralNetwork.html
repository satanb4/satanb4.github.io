void NeuralNetwork::updateWeights ( )

Updates the weights of the network.

Parameters
None
Returns
None

The weights are updated using the delta and the learning rate

Updating the weights for the hidden layers If the layer is the output layer, the bias is not updated, number of neurons = number of columns If the layer is a hidden layer, the bias is updated, number of neurons = number of columns - 1

The weight is updated by the product of the learning rate, the delta, the derivative of the activation function and the neuron value
